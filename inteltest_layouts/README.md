# Directory Overview

This directory contains layout images of **100 test cases** intended for validation. Each test case is annotated with placement constraints, and the optimal metrics for each are documented in the [`intel_testsuite.md`](#intel_testsuite.md) file located in the main repository.

## Contents

- **Layout Images:** 
  - 100 test case images for validation
  - Each annotated with placement constraints

- **Optimal Metrics:** 
  - Found in [`intel_testsuite.md`](#intel_testsuite.md)

- **Test Dataset:** 
  - Located at [`data/PrimeTensorDataTest.tar.gz`](#data/PrimeTensorDataTest.tar.gz)

- **Data Loader:** 
  - Available in [`testdataloader.py`](#testdataloader.py)
